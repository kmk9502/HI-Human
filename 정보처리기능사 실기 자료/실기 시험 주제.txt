1.응용SW기초 기술 활용

    1) 운영체제 기초 활용

	- 운영체제의 특징 파악
	- 운영체제의 기본 명령어를 활용하여 실무작업 처리
	- 운영체제의 핵심기능을 파악하여 실무작업 수행

    2) 데이터베이스 기초 활용

	- 관계형 데이터베이스의 개념 이해 및 실무 활용
	- 데이터베이스 관리에 관한 개념을 이해하고 실무 작업 수행

    3) 네트워크 기초 활용

	- 네트워크 계층 구조를 파악
	- 네트워크 프로토콜을 파악

2. 프로그래밍 언어 활용

    1) 알고리즘 구현

	- 절차형 프로그래밍 언어로 알고리즘을 처리할 수 있다. (C언어)
	- 객체지향형 프로그래밍 언어로 알고리즘을 처리할 수 있다.(JAVA 언어)

    2) 프로그래밍 언어 활용

	- 프로그래밍 언어의 기본 문법을 이해하고 언어의 특성을 활용하여 프로그래밍 
	  작업을 수행할 수 있다.
	- 라이브러리를 활용하여 프로그래밍 작업을 수행할 수 있다.

3. 애플리케이션 테스트 수행

    1) 애플리케이션 테스트 수행

	- 애플리케이션의 테스트 개념을 이해하고 실무작업 수행
	- 애플리케이션의 결함관리 개념 이해 및 실무작업 수행

    2) 애플리케이션 결함 조치

	- 애플리케이션의 결함조치의 우선순위를 이해하고 결정할 수 있다.
	- 애플리케이션의 결함 조치 관리를 이해하고 실무작업 수행 가능

4. SQL 활용

    1) 기본 SQL 작성하기

	- DDL문을 활용하여 기본 SQL을 작성할 수 있다.
	- DML문을 활용하여 기본 SQL을 작성할 수 있다.
	- DCL문을 활용하여 기본 SQL을 작성할 수 있다.

    2) 고급 SQL 작성하기

 	- 인덱스를 활용한 쿼리를 작성할 수 있다.
	- 뷰를 활용하여 쿼리를 처리할 수 있다.
	- 다중테이블 검색을 할 수 있다.
