문제 : 2진수의 값을 10진수로 변환하는 프로그램. 순서도 빈칸과 c언어 코드 실행결과 적기.

    > 순서도

	시작 -> b[5] -> n = 0 -> i=0, 4, 1 (빈칸) -> n -> 종료

    > c언어 코드

	#include<stdio.h>
	#define K 5	// 전처리문, K 변수는 항상 정수 5로 유지

	int main()
	{
		int n = 0;
		int b[K] = {1, 0, 1, 1, 0}, temp; // 5칸(0~4)의 배열에 1과 0을 저장
		int i, j;

		for(int i = 0; i < K; i++)
		{
			
			temp = 1;
			for(int j = 1; j <= K - 1 - i; j++) 
			{

				// 2^4(16), 2^3(8), 2^2(4), 2^1(2), 1

				temp = temp * 2;

			}

			//  i가 4가 될 때 temp는 반복문이 실행되지 않아 temp = 1 이다.

			n = n + temp * b[i];
		}
		printf("%d",n);
	}




답 : n = n + b[i] * 2^(4-i)

    순서도 빈칸 : 2진수를 10진수로 변환하는 코드를 한 줄로 적어야 한다.

	b[5] = {1,0,1,1,0};
	i=0, 4, 1 : for(int i=0; i < 4; i++)

	n = n + b[i] * 2^(4-i)

    
    c언어 코드 실행결과 : 16 + 4 + 2 = 22
