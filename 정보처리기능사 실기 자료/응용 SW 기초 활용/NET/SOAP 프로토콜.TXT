문제 : 다음에서 설명하는 빈칸에 알맞은 프로토콜을 작성하시오.

    웹 서비스 방식에서 파일 전송은 HTTP 기반의 (    )을 사용하여 송수신한다. (    ) 대신 레스트 폴(RESful)
    프로토콜로 대체할 수 있다.

답 : SOAP (Simple Object Access Protocol)

    - HTTP, HTTPS, SMTP 등을 통해 XML 기반 메시지를 컴퓨터 네트워크 상에서 교환하는 프로토콜

    - 기존의 원격 기술에 프록시나 방화벽에 구애 받지 않음

    - 에러 처리가 기본적으로 내장되어 있음

    - 분산 환경에서 사용하기 적합

    - RESTful에 비해 상대적으로 무거우며 속도도 느림
